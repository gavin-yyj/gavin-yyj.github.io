Ctrl

快捷键 名称 效果
Ctrl+F Find… 在当前文件进行文本查找
Ctrl+R Replace… 在当前文件进行文本替换
Ctrl+Z Undo 撤销
Ctrl+Y Delete Line 删除光标所在行 或 删除选中的行
Ctrl+X Cut 剪切光标所在行 或 剪切选择内容
Ctrl+C 或 Ctrl+Instance Copy 复制光标所在行 或 复制选择内容
Ctrl+D Duplicate Line or Selection 复制光标所在行 或 复制选择内容，并把复制内容插入光标位置下面
Ctrl+W Extend Selection 递进式选择代码块。可选中光标所在的单词或段落，连续按会在原有选中的基础上再扩展选中范围
Ctrl+E Recent Files 显示最近打开的文件记录列表
Ctrl+N Class… 根据输入的 类名 查找类文件
Ctrl+J Insert Live Template.. 插入自定义动态代码模板
Ctrl+P Parameter Info 方法参数提示显示
Ctrl+Q Quick Documentation 光标所在的变量 / 类名 / 方法名等上面（也可以在提示补充的时候按），显示文档内容
Ctrl+U Super Method 前往当前光标所在的方法的父类的方法 / 接口定义
Ctrl+B 或 Ctrl+鼠标左键 Declaration 进入光标所在的方法/变量的接口或是定义处
Ctrl+H Type Hierachy 显示当前类的层次结构
Ctrl+o Override Methods.. 选择可重写的方法
Ctrl+I Implement Methods… 选择可继承的方法
Ctrl++ Expand 展开代码
Ctrl+- Collapse 折叠代码
Ctrl+/ Comment With Line Comment 注释光标所在行代码，会根据当前不同文件类型使用不同的注释符号
Ctrl+F1 Error Description 在光标所在的错误代码处显示错误信息
Ctrl+F3 Find Word at Caret 跳转到所选中的词的下一个引用位置
Ctrl+F11 Toggle Bookmark with Mnemonic 选中文件 / 文件夹，使用助记符设定 / 取消书签
Ctrl+1,2,3,4... Go to Bookmark 定位到对应数值的书签位置
Ctrl+Tab Select Opposite Diff Pane 编辑窗口切换，如果在切换的过程又加按上delete，则是关闭对应选中的窗口
Ctrl+Backspace Basic 基础代码补全，默认在 Windows 系统上被输入法占用，需要进行修改，建议修改为 Ctrl + 逗号
Ctrl+\ Delete to Word Start 删除光标前面的单词或是中文句
Ctrl+Delete Delete to Word End 删除光标后面的单词或是中文句
Ctrl+鼠标左键 在打开的文件标题上，弹出该文件路径
Ctrl+光标定位 按 Ctrl 不要松开，会显示光标所在的类信息摘要
Ctrl+左方向键 Move Caret to Previous Word 光标跳转到当前单词 / 中文句的左侧开头位置
Ctrl+右方向键 Move Caret to Next Word 光标跳转到当前单词 / 中文句的右侧开头位置
Ctrl+上方向键 Scroll Up 等效于鼠标滚轮向前效果
Ctrl+下方向键 Scroll Down 等效于鼠标滚轮向后效果

Alt

快捷键 名称 效果
Alt+Q Context info 弹出一个提示，显示当前类的声明 / 上下文信息
Alt+F1 Select in.. 显示当前文件选择目标弹出层，弹出层中有很多目标可以进行选择
Alt+F7 Find Usages in File 查找光标所在的方法 / 变量 / 类被调用的地方
Alt+F8 Evaluate Expression 在 Debug 的状态下，选中对象，弹出可输入计算表达式调试框，查看该输入内容的调试结果
Alt+Home 定位 / 显示到当前文件的 Navigation Bar
Alt+Enter Show Intension Actions IntelliJ IDEA 根据光标所在问题，提供快速修复选择，光标放在的位置不同提示的结果也不同
Alt+Insert Generate 代码自动生成，如生成对象的 set / get 方法，构造函数，toString() 等
Alt+左方向键 Select Previous Tab 切换当前已打开的窗口中的子视图，比如Debug窗口中有Output、Debugger等子视图，用此快捷键就可以在子视图中切换
Alt+右方向键 Select Next Tab 按切换当前已打开的窗口中的子视图，比如Debug窗口中有Output、Debugger等子视图，用此快捷键就可以在子视图中切换
Alt+上方向键 Previous Method 当前光标跳转到当前文件的前一个方法名位置
Alt+下方向键 Next Method 当前光标跳转到当前文件的后一个方法名位置
Alt+1,2,3,4... 显示对应数值的选项卡，其中 1 是 Project 用得最多

Shift

快捷键 名称 效果
Shift+F6 Rename 对文件 / 文件夹 重命名
Shift+F7 Smart step into 在 Debug 模式下，智能步入。断点所在行上有多个方法调用，会弹出进入哪个方法
Shift+F8 Step Out 在 Debug 模式下，跳出
Shift+F9 Debug 等效于点击工具栏的 Debug 按钮
Shift+F10 Run 等效于点击工具栏的 Run 按钮
Shift+F11 Show Bookmarks 弹出书签显示层
Shift+Tab 取消缩进
Shift+End 选中光标到当前行尾位置
Shift+Home 选中光标到当前行首位置
Shift+Enter Start New Line 开始新一行。光标所在行下空出一行，光标定位到新行位置
Shift+滚轮前后滚 当前文件的横向滚动轴滚动

Ctrl + Alt

快捷键 名称 效果
Ctrl + Alt + L Reformat Code 格式化代码，可以对当前文件和整个包目录使用。（有些文件格式化是有特殊要求的，比如 SQL 文件必须设置方言才可被格式化）
Ctrl + Alt + O Optimise Imports 优化导入的类，可以对当前文件和整个包目录使用
Ctrl + Alt + T Surroud with 对选中的代码弹出环绕选项
Ctrl + Alt + H Call Hierarchy 调用层次
Ctrl + Alt + B implementation 在某个调用的方法名上使用会跳到具体的实现处，可以跳过接口
Ctrl + Alt + S Settings 打开 IntelliJ IDEA 系统设置
Ctrl + Alt + F7 Show Usages 显示使用的地方。寻找被该类或是变量被调用的地方，用弹出框的方式找出来
Ctrl + Alt + Enter Start New Line Before Current 光标所在行上空出一行，光标定位到新行
Ctrl + Alt + 左方向键 back 退回到上一个操作的地方
Ctrl + Alt + 右方向键 forward 前进到上一个操作的地方

Ctrl + Shift

快捷键 名称 效果
Ctrl + Shift + F Find in Path 根据输入内容查找整个项目 或 指定目录内文件
Ctrl + Shift + R Replace in Path 根据输入内容替换对应内容，范围为整个项目 或 指定目录内文件
Ctrl + Shift + J Join Line 自动将下一行合并到当前行末尾
Ctrl + Shift + Z Redo 取消撤销
Ctrl + Shift + W Shrink Selection 递进式取消选择代码块。可选中光标所在的单词或段落，连续按会在原有选中的基础上再扩展取消选中范围
Ctrl + Shift + N File.. 通过文件名定位 / 打开文件 / 目录，打开目录需要在输入的内容后面多加一个正斜杠
Ctrl + Shift + U Toggle Case 对选中的代码进行大 / 小写轮流转换
Ctrl + Shift + T Test 对当前类生成单元测试类，如果已经存在的单元测试类则可以进行选择
Ctrl + Shift + C Copy Paths 复制当前文件磁盘路径到剪贴板
Ctrl + Shift + H Method Hierarchy 显示方法层次结构
Ctrl + Shift + B 或 Ctrl + Shift + 左键单击 Type Declaration 跳转到类型声明处
Ctrl + Shift + I Quick Definition 快速查看光标所在的方法 或 类的定义
Ctrl + Shift + / Comment with Block Comment 代码块注释
Ctrl + Shift + [ Move Caret to Code Block Start with Selection 选中从光标所在位置到它的顶部大括号位置
Ctrl + Shift + ] Move Caret to Code Block End with Selection 选中从光标所在位置到它的底部大括号位置
Ctrl+Shift+ + Expand all 展开所有代码
Ctrl+Shift+ - Collapse all 折叠所有代码
Ctrl + Shift + F7 Highlight Usages in File 高亮显示所有该选中文本，按Esc高亮消失
Ctrl + Shift + F8 View Breakpoints 在 Debug 模式下，查看所有断点
Ctrl + Shift + F12 Hide All Tool Windows 编辑器最大化
Ctrl + Shift + space smartType 智能代码提示
Ctrl + Shift + Enter Complete Current Statement 自动结束代码，行末自动添加分号
Ctrl + Shift + 1,2,3,4.. Toggle Bookmark 1,2,3,4.. 快速添加指定数值的书签
trl + Shift + 左方向键 Move Caret to Previous Word with Selection 在代码文件上，光标跳转到当前单词 / 中文句的左侧开头位置，同时选中该单词 / 中文句
Ctrl + Shift + 右方向键 Move Caret to Next Word with Selection 在代码文件上，光标跳转到当前单词 / 中文句的右侧开头位置，同时选中该单词 / 中文句
Ctrl + Shift + 前方向键 Move Statement Up 光标放在方法名上，将方法移动到上一个方法前面，调整方法排序
Ctrl + Shift + 后方向键 Move Statement Down 光标放在方法名上，将方法移动到下一个方法前面，调整方法排序

Alt + Shift

快捷键 名称 效果
Alt + Shift + N Open Task.. 选择 / 添加 task
Alt + Shift + F7 Force Step Into 在 Debug 模式下，下一步，进入当前方法体内，如果方法体还有方法，则会进入该内嵌的方法中，依此循环进入
Alt + Shift + 左键双击 Add or Delete Caret 选择被双击的单词 / 中文句，按住不放，可以同时选择其他单词 / 中文句
Alt + Shift +前方向键 Back 移动光标所在行向上移动
Alt + Shift + 后方向键 Forward 移动光标所在行向下移动

Ctrl + Shift + Alt

快捷键 名称 效果
Ctrl + Shift + Alt + V Paster simple 无格式粘贴
Ctrl + Shift + Alt + N Symbol 前往指定的变量 / 方法
Ctrl + Shift + Alt + S Project Structure 打开当前项目设置
Ctrl + Shift + Alt + 按下鼠标左键滑动 可以选择一个矩形框，然后能对这个矩形框同时进行修改

其他

快捷键 名称 效果
F2 Next Highlight Error 跳转到下一个高亮错误 或 警告位置
F3 Find Next 在查找模式下，定位到下一个匹配处
F4 Jump to Source 编辑源
F7 Step Into 在 Debug 模式下，进入下一步，如果当前行断点是一个方法，则进入当前方法体内，如果该方法体还有方法，则不会进入该内嵌的方法中
F8 Step Over 在 Debug 模式下，进入下一步，如果当前行断点是一个方法，则不进入当前方法体内
F9 Resume Program 在 Debug 模式下，恢复程序运行，但是如果该断点下面代码还有断点则停在下一个断点上
F11 Toggle Bookmark 添加书签
F12 Jump to last Tool Window 回到前一个工具窗口
ESC Escape 从工具窗口进入代码文件窗口
连按两次Shift 弹出 Search Everywhere 弹出层

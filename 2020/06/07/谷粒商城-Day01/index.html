4. 通用Mapper

导入通用Mapper依赖

<!-- 通用mapper -->
<dependency>
    <groupId>tk.mybatis</groupId>
    <artifactId>mapper-spring-boot-starter</artifactId>
    <!--谷粒学院上面是1.2.3版本，太老了，运行会报错，用新版本-->
	<version>2.1.5</version>
    <exclusions>
        <exclusion>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-jdbc</artifactId>
        </exclusion>
    </exclusions>
</dependency>

其他注意事项：

如果之前用的不是通用Mapper，并且已经配置过，那么记得将原先的用到Mapper的地方的导包改成tk.mybatis，而不能用org.mybatis。

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;
import tk.mybatis.spring.annotation.MapperScan;//注意

@MapperScan(basePackages = "com.gavin.gmall.user.mapper")
@SpringBootApplication
public class GmallUserApplication {

    public static void main(String[] args) {
        SpringApplication.run(GmallUserApplication.class, args);
    }
}

另外其他用到Mapper的地方继承Mapper的时候也要选择tk.mybatis包下的mapper。
